# stock-market-prediction-dashboard
The project implements a hybrid stock prediction system that combines numerical data from stock price history with textual sentiment extracted from financial news. The system uses APIs for data collection, applies sentiment analysis using NLP techniques, and generates predictions based on sentiment scores. 

## Features
- Real-time stock data
- News sentiment analysis
- Stock prediction
- Top gainers & losers

## Tech Stack
Python, Streamlit, NLP, yfinance

## Run Locally
pip install -r requirements.txt
streamlit run app.py
